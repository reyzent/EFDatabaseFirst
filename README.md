# EF Database First

Подход Database First в Entity Framework 6.

## Описание
Проект демонстрирует подход Database First:
- Создание моделей из существующей базы данных
- Использование EF Designer
- Автоматическая генерация классов из БД

## Технологии
- .NET Framework 4.7.2
- Entity Framework 6
- SQL Server
- ADO.NET Entity Data Model

## Особенности
- Модели созданы через EF Designer from database
- Визуальное представление сущностей
- Автоматическая генерация контекста и классов
- Использование ObjectContext/DbContext
